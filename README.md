# 看板项目原型

## 项目目录结构
```
├─kanban-app 看板后端开发根目录
│  └─src
│      ├─main
│      │  ├─java        后端源码目录
│      │  ├─resources   后端资源
│      │  └─webapp      后端网站根目录
│      └─test
│          └─java       测试源码目录
│
└─kanban-resources  看板前端开发根目录
    ├─js    前端js源码目录
    ├─less  前端样式存放目录，存放.less文件
    ├─libs  第三方js,css类库，使用bower管理
    └─views 看板界面存放目录
```
